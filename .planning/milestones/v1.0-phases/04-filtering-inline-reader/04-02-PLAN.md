| Nostr relay -> browser (PRIMARY) | `Article.content` is author-controlled raw Markdown from arbitrary pubkeys; rendered to DOM via react-markdown — the phase's chief threat surface |
| Markdown body -> external sites | Author-supplied links/images point to arbitrary URLs the user may click/load |
| npm registry -> build | New dependencies (react-markdown, remark-gfm, rehype-sanitize) enter the supply chain |

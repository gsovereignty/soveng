Scaffold the Soveng SPA from scratch (greenfield repo, no package.json or src/ yet) and deliver the first real, visible UI: a terminal-themed green-phosphor boot sequence that resolves to a blinking prompt, running in the local dev server.

This is the first task of the Walking Skeleton. Output: A buildable Vite + React 19 + TypeScript + shadcn/ui (new-york, Tailwind v4) project that renders the terminal boot-sequence placeholder locally. Deployment is handled by Plan 02 (Wave 2).
</objective>
